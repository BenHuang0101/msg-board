# 🧾 Message Board (Spring Boot + Vue3 + MySQL + Docker)

## ⚙️ 環境需求

- Node.js >= 18
- Java >= 21
- Docker + Docker Compose

## 🏁 啟動方式

```bash
# clone 專案
git clone https://github.com/xxx/msg-board.git
cd msg-board

# 複製範例 env 檔
cp .env.example .env
# → 請填入你的密碼等資訊

# 啟動所有服務
docker compose up --build
```
